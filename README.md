# Rkisoner

Rkisoner is a Chrome extension that helps users control YouTube content
by blurring videos based on selected tags.  
The goal is to improve focus and reduce unwanted content exposure.

## Features
- Tag-based video blurring
- Simple and lightweight
- Built for productivity and content control
